## 문제 : 15663번 N과 M (9)

### 입력 
1. N, M 
- 공백으로 구분 
- 1 ≤ M ≤ N ≤ 8
2. N개의 수 
- 공백으로 구분 
- 10,000보다 작거나 같은 자연수

### 조건 
1. 길이가 M인 수열을 모두 구하는 프로그램
2. N개의 자연수 중에서 M개를 고른 수열

### 출력 
1. 한 줄에 하나씩 조건을 만족하는 수열 출력 
- 중복 x 
- 공백으로 구분 
- 수열은 오름차순으로 출력

```java
import java.util.*;
import java.io.*;

public class Main {
  static int n, m;
  static int[] arr, selected;
  static boolean[] used; 
  static StringBuilder sb = new StringBuilder(); 
  public static void main(String[] args)throws IOException {
    BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

    StringTokenizer st = new StringTokenizer(br.readLine());
    

    // N, M 입력
    n = Integer.parseInt(st.nextToken());
    m = Integer.parseInt(st.nextToken()); 

    arr = new int[n]; 
    selected = new int[m]; 
    used = new boolean[n];

    // N개의 수 입력 
    st = new StringTokenizer(br.readLine());
    for(int i = 0; i < n; i++) {
      arr[i] = Integer.parseInt(st.nextToken()); 
    }
    br.close(); 

    // 오름차순 정렬
    Arrays.sort(arr);

    dfs(0);

    System.out.println(sb);

    
  }

  static void dfs(int depth) {
    // 선택한 숫자의 개수가 m과 같으면 리턴 
    if(depth == m) {
      for(int i : selected) {
        sb.append(i).append(' ');
      }
      sb.append('\n');
      return; 
    }

    // 같은 depth에서 이전에 사용한 숫자를 저장해 중복 제거 
    int forward = Integer.MIN_VALUE;

    for(int i = 0; i < n; i++) {
      // 사용한 숫자 중복 처리 
      if(used[i]) {
        continue;
      }

      if(arr[i] == forward) {
        continue;
      }

      // 사용 표시 
      used[i] = true; 
      
      selected[depth] = arr[i];
      forward = arr[i];

      // 다음 depth로 재귀 호출 
      dfs(depth + 1);

      // 사용 표시 해제 
      used[i] = false; 
    }
  }
}
```
