## 문제 : 11725번 트리의 부모 찾기 

### 입력 
1. 노드의 개수 N
- 2 ≤ N ≤ 100,000
2. N-1줄에 걸쳐 트리 상에 연결된 두 정점 
- 공백으로 구분 

### 조건 
- 루트 없는 트리가 주어짐
- 루트는 1 
- 각 노드의 부모를 구하는 프로그램 작성 

### 출력 
1. 한 줄에 하나씩 N-1줄에 걸쳐 각 노드의 부모 노드 번호 출력


```java
import java.util.*;
import java.io.*;

public class Main {
  static int n;
  static List<Integer>[] listGraph; 
  static int[] parent; 
  static boolean[] visited; 
  static StringTokenizer st; 
  
  public static void main(String[] args)throws IOException {
    BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

    // 노드의 개수 N
    n = Integer.parseInt(br.readLine());

    // 인접 리스트 초기화 
    listGraph = new ArrayList[n+1];

    for(int i = 1; i <= n; i++)  {
      listGraph[i] = new ArrayList<>();  
    }

    // 트리 상 연결된 두 정점     
    for(int i = 0; i < n-1; i++) {
      st = new StringTokenizer(br.readLine());
      int u = Integer.parseInt(st.nextToken());
      int v = Integer.parseInt(st.nextToken()); 

      // 양방향 그래프 
      listGraph[u].add(v);
      listGraph[v].add(u); 
    }

    // 부모 배열 초기화 
    parent = new int[n+1];
    // 방문 여부 확인 배열 초기화  
    visited = new boolean[n+1]; 

    // bfs 루트 1번 부터 실행 
    bfs(1);     

    // 부모 노드 번호 출력 
    StringBuilder sb = new StringBuilder(); 
    for(int i = 2; i <= n; i++) {
      sb.append(parent[i]).append('\n');
    }
    System.out.print(sb); 
  }

  static void bfs(int root) {
    Queue<Integer> queue = new LinkedList<>(); 

    // 루트 노드 방문 체크 
    visited[root] = true; 
    queue.offer(root);

    while(!queue.isEmpty()) {
      int current = queue.poll();

      // 인접 노드 순회 
      for(int next: listGraph[current]) {
        if(!visited[next]) {
          visited[next] = true; 

          // 다음 노드의 부모는 현재 노드 
          parent[next] = current;
          queue.offer(next); 
        }
      }
    }
  }
}
```
