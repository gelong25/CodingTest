## 문제 : 10994번 별 찍기 -19

### 입력
1. 정수 N
- 1 ≤ N ≤ 100

### 조건 
- 예제를 보고 규칙을 유추해 별 찍기 

### 예제 1
```shell
*
```

### 예제 2 
```shell
*****
*   *
* * *
*   *
*****
```

### 예제 3 
```shell
*********
*       *
* ***** *
* *   * *
* * * * *
* *   * *
* ***** *
*       *
*********
```

### 출력 
1. 첫째 줄 부터 차례로 별 출력 

```java
import java.io.*;
import java.util.Scanner;

public class Main {
  static char[][] stars;

  public static void main(String[] args) throws IOException {
    Scanner sc = new Scanner(System.in); 
    int n = sc.nextInt();

    if (n == 1) {
      System.out.println("*");
    } else {
      int size = 4 * (n - 1) + 1;
      stars = new char[size][size];

      // 공백으로 초기화
      for (int i = 0; i < size; i++) {
        for (int j = 0; j < size; j++) {
          stars[i][j] = ' ';
        }
      }

      // 재귀 함수 호출 
      drawStar(0, n);

      // 결과 출력
      StringBuilder sb = new StringBuilder();
      for (int i = 0; i < size; i++) {
        for (int j = 0; j < size; j++) {
          sb.append(stars[i][j]);
        }
        sb.append('\n');
      }
      System.out.print(sb.toString());
    }
  }

  static void drawStar(int start, int level) {
    if (level <= 0) return;

    int len = 4 * (level - 1) + 1;
    int end = start + len - 1;

    // 테두리 그리기
    for (int i = start; i <= end; i++) {
      stars[start][i] = '*';       
      stars[end][i] = '*';          
      stars[i][start] = '*';       
      stars[i][end] = '*';         
    }

    // 다음 테두리 그리기 
    drawStar(start + 2, level - 1);
  }
}

```
