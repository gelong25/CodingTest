## 문제 : 11660번 구간합 구하기 5

### 입력
1. 표의 크기 N, 합을 구해야 하는 횟수 M
- 공백으로 구분 
- 1 ≤ N ≤ 1024
- 1 ≤ M ≤ 100,000
2. N개의 줄에 걸쳐 표에 채워 져야 하는 수
- 공백으로 구분 
- 1,000보다 작거나 같은 자연수
3. M개의 줄에 걸쳐 네 개의 정수 x1, y1, x2, y2
- 공백으로 구분
- x1 ≤ x2, y1 ≤ y2 

### 조건 
- N x N 표가 주어졌을 때 (x1, y1)부터 (x2, y2)까지 합을 구하는 프로그램

### 출력 
1. M개의 줄에 걸쳐 (x1, y1)부터 (x2, y2)까지 합 출력

```java
import java.util.*;
import java.io.*;

public class Main {
  public static void main(String[] args)throws IOException {
    BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

    StringTokenizer st = new StringTokenizer(br.readLine()); 

    // 표의 크기 n 
    int n = Integer.parseInt(st.nextToken());
    // 합을 구해얄 하는 횟수 m 
    int m = Integer.parseInt(st.nextToken()); 

    // dp 배열 
    int[][] dp = new int[n+1][n+1];

    for(int i = 1; i <= n; i++) {
      st = new StringTokenizer(br.readLine());
      // 각 행의 누적합 계산 
      for(int j = 1; j <= n; j++) {
        dp[i][j] = dp[i][j-1] + Integer.parseInt(st.nextToken());
      }
    }

    StringBuilder sb = new StringBuilder(); 

    for(int i = 1; i <= m; i++) {
      // 구간합을 구해야 할 위치 입력 
      st = new StringTokenizer(br.readLine());

      int x1 = Integer.parseInt(st.nextToken());
      int y1 = Integer.parseInt(st.nextToken());
      int x2 = Integer.parseInt(st.nextToken());
      int y2 = Integer.parseInt(st.nextToken()); 

      // 누적합 저장 변수 
      int prefixSum = 0;

      // 누적합 계산 
      for(int j = x1; j <= x2; j++) {
        prefixSum = prefixSum + (dp[j][y2] - dp[j][y1-1]);    
      }

      sb.append(prefixSum + "\n"); 
      
    }

    System.out.print(sb); 

    
  }
}
```
