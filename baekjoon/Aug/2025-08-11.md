## 문제 : 25943번 양팔저울

### 입력
1. 자갈 개수 n
- 2 <= n <= 10,000
2. n개의 줄에 걸쳐 자갈의 무게
- 공백으로 구분 
- 1 <= 자갈의 무게 
- 자갈 무게의 총 합:  10,000,000 이하 

### 조건
- 입력 받은 자갈을 아래 규칙대로 올렸을 때 평형을 맞추는 데 필요한 무게추의 최소 개수를 구하는 프로그램
- 1번 자갈을 왼쪽에 올림 
- 2번 자갈을 오른쪽에 올림 
- i=3, ..., n번 자갈
  - 양팔 저울이 평형일 경우 i번 자갈을 왼쪽에 올림
  - 양팔 저울이 평형이 아닐 경우 가벼운 쪽에 올림 
- 무게추: 1g, 2g, 5g, 10g, 20g, 50g, 100g
  
### 출력 
1. 평형을 맞추는데 추가적으로 필요한 무게추의 최소 개수

```java
import java.util.*;
import java.io.*;

public class Main {
  public static void main(String[] args)throws IOException {
    BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

    // 자갈의 개수 n
    int n = Integer.parseInt(br.readLine());

    // n개의 수 입력 
    StringTokenizer st = new StringTokenizer(br.readLine());
    long left = Long.parseLong(st.nextToken());
    long right = Long.parseLong(st.nextToken());

    for(int i = 3; i <= n; i++) {
      long weight = Long.parseLong(st.nextToken()); 

      // 무게가 같거나 왼쪽이 무거운 경우 오른쪽에 무게 추가 
      if(left <= right) {
        left += weight; 
      } else {
        right += weight;  // 그 외엔 왼쪽에 무게 추가 
      }
    }

    // 왼쪽과 오른쪽 무게 차이의 절댓값
    long comp = Math.abs(left - right); 

    // 차이가 없을 경우 0 출력
    if(comp == 0) {
      System.out.println(0);
      return; 
    }

    // 무게추 배열
    int[] weights = {100, 50, 20, 10, 5, 2, 1}; 

    // 정답을 저장할 변수 
    long answer = 0; 
    
    for(int num : weights) {
      if(comp == 0) {
        break; 
      }
      // 현재 무게 추로 더할 수 있는지 계산 
      answer += (comp / num); 
      // 남은 무게 갱신
      comp %= num; 
    }
    System.out.println(answer); 
    
  }
}
```
